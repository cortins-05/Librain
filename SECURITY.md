# Security Policy

## Supported versions

This project is a HackUDC hackathon build. We aim to keep `main` in a working state and accept security fixes via pull requests.

## Reporting a vulnerability

Please report security issues **privately** via email:

- **Contact:** lucasortins@gmail.com
- **Subject:** `[SECURITY] Librain vulnerability report`

Include:

- A clear description of the issue
- Steps to reproduce (or PoC)
- Impact assessment (what an attacker can do)
- Any suggested fix (if you have one)

## Disclosure policy

- We will acknowledge receipt as soon as possible.
- We will work on a fix and coordinate a disclosure date if needed.
- Please do not publicly disclose the issue before a fix is available.
