# Support

## I found a bug / I have a feature idea

Please open a GitHub Issue with:

- What you expected
- What happened
- Steps to reproduce
- Logs/screenshots if relevant

Tip: if you're new, look for issues labeled **good first issue**.

## Security issues

Please do **not** open a public issue for security vulnerabilities.  
Follow `SECURITY.md` and email: **lucasortins@gmail.com**
